--> Before commiting a branch you have to specify user name and user eamilid using below given commands.
git config --global user.name "your name"
to check your name
git config --global user.name

git config --global user.email "your name"
to check your email
git config --global user.email

--> To merge your code from one branch to another new branch
1) git merge {new branch}

--> Cloning your work into repositories
2) git remote add origin {clone repo URL Link} 
3) git commit -m "{your message}"
4) git push origin {your branch name}



--> To upload your work on your github branch
5) git push -u origin {your branch name}

--> Now if want to go to your previous code
6) git log
7) git checkout {hash code}

--> To clone some project work in our system
8) git clone {link of other project repositories}

--> To rename your branch
9) git branch -m {old-name new-name}
Ex:-- git branch -m pruthvi pruthvi1
